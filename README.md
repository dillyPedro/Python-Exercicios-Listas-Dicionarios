📊 Exercícios em Python: Listas e Dicionários

Este repositório contém uma série de atividades desenvolvidas em Python com o objetivo de praticar lógica de programação, manipulação de estruturas de dados (listas e dicionários), controle de fluxo e tratamento de regras de negócio.

🚀 Atividades desenvolvidas

👥 Análise de Recursos Humanos:
Programa focado em inteligência de negócios que:
Consolida dados de idades de colaboradores de múltiplos setores em um dicionário, calcula a média de idade específica de cada setor de forma dinâmica, determina a média de idade geral de toda a empresa e filtra a quantidade exata de pessoas que estão acima dessa média geral.

💡 Conceitos utilizados:
- Manipulação de dicionários com listas aninhadas;
- Métodos .values(), .keys() e .extend();
- Funções sum() e len();
- Consolidação e filtragem de dados;

📅 Validação de Datas:
Programa que analisa a consistência de uma linha temporal, onde:
Recebe valores inteiros para dia, mês e ano, valida os limites reais de meses e anos, determina o limite máximo de dias para cada mês do calendário e aplica a lógica matemática de anos bissextos para validar o mês de fevereiro (casos de borda).

💡 Conceitos utilizados:
- Estruturas condicionais encadeadas if, elif, else;
- Operadores lógicos compostos and, or`;
- Operador módulo % para cálculo de ano bissexto;
- Validação e consistência de dados de entrada;

🧫 Estudo de Crescimento de Bactérias:
Programa voltado para análise de dados e biologia computacional que:
Percorre uma lista de dados históricos de crescimento, calcula o percentual de crescimento diário comparando a amostra atual com a do dia anterior através de uma equação matemática e trata possíveis erros de divisão por zero na base de dados.

💡 Conceitos utilizados:
- Iteração baseada em índices com range(1, len(...));
- Manipulação e atualização dinâmica de listas com .append();
- Prevenção de erros em tempo de execução (tratamento de divisão por zero);
- Modelagem de fórmulas matemáticas em código;

🧠 Objetivo
Essas atividades foram desenvolvidas com foco em:
Fortalecer a manipulação de estruturas de dados complexas, praticar a limpeza e preparação de dados, tratar regras de negócio do mundo real e desenvolver soluções eficientes com códigos limpos e legíveis.

🛠️ Tecnologias utilizadas:
- Python 3
